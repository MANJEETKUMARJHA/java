## 📚 Topics Covered

### 🟢 Beginner Level (Core Basics)
- **Introduction to Java:** Setup, JDK, JRE, JVM, and the structure of a Java program.
- **Variables & Data Types:** Primitives (int, double, char, boolean, etc.) and Non-Primitives (String, Arrays).
- **Operators:** Arithmetic, Relational, Logical, and Bitwise operators.
- **Control Flow Statements:** `if`, `else`, `switch`, `for`, `while`, and `do-while` loops.
- **Methods/Functions:** Declaration, parameters, return types, variable scopes, and method overloading.

### 🟡 Intermediate Level (Object-Oriented Programming & Core APIs)
- **Object-Oriented Programming (OOPs):**
  - Classes & Objects
  - Inheritance
  - Polymorphism (Overloading & Overriding)
  - Encapsulation
  - Abstraction (Abstract Classes & Interfaces)
- **String Handling:** Deep dive into `String`, `StringBuilder`, and `StringBuffer`.
- **Exception Handling:** `try`, `catch`, `finally`, `throw`, and `throws`, custom exceptions.
- **File Handling (I/O):** Reading and writing to files using `java.io` (Streams) and `java.nio` (Paths, Files).
- **Collections Framework:** `List`, `Set`, `Map`, `Queue`, and their implementations (ArrayList, HashMap, HashSet, LinkedList, etc.).

### 🔴 Advanced Level (Modern Java & Multithreading)
- **Multithreading & Concurrency:** Thread creation, synchronization, `volatile`, `wait/notify`, and the `java.util.concurrent` package (Executors, CompletableFuture).
- **Generics:** Creating type-safe classes, methods, and wildcards.
- **Java 8+ Features:**
  - Lambda Expressions & Functional Interfaces
  - Stream API for data processing
  - Optional Class (handling nulls)
  - Default & Static methods in Interfaces
  - New Date and Time API (`java.time`)
  - Records and Pattern Matching (Java 14+)
- **Design Patterns in Java:** Singleton, Factory, Observer, Builder, and Strategy patterns.
- **JDBC (Java Database Connectivity):** Connecting Java applications to relational databases (MySQL, PostgreSQL).
- **JVM Internals & Garbage Collection:** Understanding memory management, Heap, Stack, and GC algorithms.

## 🚀 How to Use This Repository
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/MANJEETKUMARJHA/java.git
   ```
2. Navigate to the specific topic directory you want to study.
3. Open the folder in your favorite IDE.
4. Read the comments in the source code to understand the concepts.
5. Compile and run the Java files from the terminal or directly in your IDE:
   ```bash
   javac FileName.java
   java FileName
   ```
6. **Experiment!** Tweak the code, break it, and fix it. That's the best way to learn.
